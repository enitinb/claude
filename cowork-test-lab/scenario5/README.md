# Scenario 5: Research Synthesis

**Test Cowork's ability to synthesize information from heterogeneous sources — market reports, news articles, analyst notes, internal chat, and customer transcripts.**

---

## What You're Testing

| Capability | Question |
|------------|----------|
| Cross-source synthesis | Does Cowork connect facts across documents that mention them differently? |
| Disagreement handling | When sources contradict (market sizing, methodology), does it reconcile rather than pick silently? |
| Mixed-register reading | Can it pull insight from both formal analyst prose and informal Slack chatter? |
| Source attribution | Does each claim trace back to a specific source? |
| Strategic translation | Does it move beyond summary to recommended action? |

---

| Folder | Task | Inputs | Time |

---

## The Insight

> "The real synthesis test is what happens when sources disagree. Bad systems pick the loudest source; good systems reconcile and call out the disagreement. Same goes for source register — pulling the strategic signal out of a Slack thread is different from pulling it out of an analyst report, and both have to land in the same brief."
