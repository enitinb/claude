# Scenario 6: Personal Knowledge Synthesis

**Test Cowork's ability to synthesize personal material — journal entries, reading highlights — into patterns and insights.**

---

## What You're Testing

| Capability | Question |
|------------|----------|
| Cross-entry pattern detection | Does Cowork surface patterns visible only across many days/sources? |
| Tone awareness | Does it read affect and tone, not just facts? |
| Restraint | Does it resist moralizing, over-claiming, or pathologizing? |
| Position-tracking | When an author *updates* their own position across sources, does it notice? |
| Specificity | Are insights grounded in what was actually written, not generic advice? |

| Folder | Task | Inputs | Time |

---

## The Insight

> "Personal-knowledge synthesis is the hardest read, not the easiest. A daily journal has more signal hidden in the gaps between entries than in any single entry. A set of reading highlights has more value in the *tensions* between sources than in any one source's argument. The test is whether Cowork can read *across* rather than read *each*."
